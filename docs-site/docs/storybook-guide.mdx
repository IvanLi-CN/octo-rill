# Storybook 导览

这里集中提供 OctoRill Storybook 的总入口，以及最常用的公开文档深链。

若你正在访问 GitHub Pages 或装配后的静态预览，请直接使用本页的 Storybook 链接。
若你正在单独运行 `docs-site`（`bun run dev` / `bun run preview`），请先在 `web/` 启动 Storybook dev server；本页链接会自动跳到对应的本地 Storybook 页面。

## 何时看 Storybook

- 想核对页面状态与占位文案时，优先看 `Pages/*` 与 `Admin/*`。
- 想确认组件密度、基础控件或版本信息回退时，优先看 `Layout/*` 与 `UI/*`。
- 想回到公开文档上下文，可使用顶部导航返回“快速开始 / 配置参考 / 产品说明”。
