# Storybook 入口

这里集中提供 OctoRill Storybook 的总入口，以及最常用的公开文档深链。若你只想直接预览完整 Storybook，请使用下面的主入口按钮。

## 何时看 Storybook

- 想核对页面状态与占位文案时，优先看 `Pages/*` 与 `Admin/*`。
- 想确认组件密度、基础控件或版本信息回退时，优先看 `Layout/*` 与 `UI/*`。
- 想回到公开文档上下文，可使用顶部导航返回“快速开始 / 配置参考 / 产品说明”。
